# Digital Catalog Generator

## When to use
- Necesitas una tienda online
- Catálogo de productos con filtros
- Sistema de carrito y checkout
- Galleries de portafolios

## Features
- Listar productos (grid responsivo)
- Filtros (categoría, precio, rango)
- Búsqueda en tiempo real
- Carrito persistente (localStorage)
- Formulario checkout
- Integrá con Stripe/PayPal (opcional)
- Responsive mobile-first

## Instructions
1. Define productos (JSON o API)
2. Especifica filtros necesarios
3. Selecciona forma de pago
4. Claude genera componentes React
5. Incluye estilos Tailwind + shadcn

## Examples
Productos: Array de objetos
```json
{"id": 1, "name": "Cocina Integral", "price": 9999, "category": "Cocinas", "image": "url"}
```

## Output
- Artifact React completo
- Componentes reutilizables
- Carrito funcional
- Filtros activos
- Responsive
